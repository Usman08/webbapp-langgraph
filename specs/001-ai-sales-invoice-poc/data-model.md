## Derived calculations (Domain layer ‚Äî .NET)

- **Quantity adjustment**: `round_half_up(prevQty √ó (1 + delta%))` ‚Üí whole units (FR-017).
- **Line total**: `Quantity √ó UnitPrice`.
- **Subtotal**: Œ£ line totals of InStock/AlternativeSuggested lines (BackOrder lines included
  only if operator retains them).
- **Discount amount**: `Subtotal √ó DiscountPercentage/100` (0 when no rule resolves).
- **Tax amount**: `(Subtotal ‚àí DiscountAmount) √ó TaxPercentage/100`.
- **Total**: `Subtotal ‚àí DiscountAmount + TaxAmount`. Rounding error tolerance ‚â§ 0.01 (SC-006).

## Seed dataset (FR-015/016)
